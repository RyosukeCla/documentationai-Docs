## Overview

U17g supports seamless integrations with popular third-party services. You can connect tools for version control, notifications, content export, and embedding to streamline your documentation processes. These integrations help automate workflows, sync content, and extend functionality without leaving your U17g workspace.

<Callout kind="tip">
Explore the integrations directory in your U17g dashboard to discover and enable available connections.
</Callout>

## Setting Up Webhooks for Notifications

Webhooks enable real-time notifications from U17g to external services. Configure them in your project settings to trigger events like page updates or new comments.

<Steps>
  <Step title="Navigate to Settings" icon="settings">
    Go to your project dashboard and select `Integrations` > `Webhooks`.
  </Step>
  <Step title="Add Webhook URL" icon="link">
    Enter your endpoint URL, such as `https://your-webhook-url.com/u17g`.
  </Step>
  <Step title="Select Events" icon="zap">
    Choose events like `page.updated` or `comment.created`.
  </Step>
  <Step title="Test and Save" icon="check-circle">
    Send a test payload and save the configuration.
  </Step>
</Steps>

### Webhook Payload Example

U17g sends JSON payloads to your endpoint. Verify your server handles `POST` requests with `Content-Type: application/json`.

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  // Example Node.js handler
  app.post('/u17g', (req, res) => {
    const { event, data } = req.body;
    console.log(`Event: ${event}`, data);
    res.status(200).json({ received: true });
  });
  ```
  ```python
  # Example Flask handler
  from flask import Flask, request, jsonify
  app = Flask(__name__)

  @app.route('/u17g', methods=['POST'])
  def webhook():
      data = request.json
      event = data.get('event')
      print(f"Event: {event}", data)
      return jsonify({'received': True})
  ```
</CodeGroup>

<ParamField path="event" param-type="string" required="true">
  The event type, e.g., `page.updated`.
</ParamField>

<ParamField body="data" param-type="object" required="true">
  Payload details including `pageId`, `title`, and `user`.
</ParamField>

## Exporting Documentation Formats

Export your U17g documentation in multiple formats for sharing or archiving.

<Tabs>
  <Tab title="PDF" icon="file">
    Generate print-ready PDFs with custom branding.
    
    ```
    curl -X POST https://api.example.com/v1/export/pdf \
      -H "Authorization: Bearer YOUR_API_KEY" \
      -d '{"projectId": "proj_123", "includeImages": true}'
    ```
  </Tab>
  <Tab title="Markdown" icon="file-text">
    Export as raw Markdown files for GitHub or static sites.
    
    ```
    curl -X POST https://api.example.com/v1/export/md \
      -H "Authorization: Bearer YOUR_API_KEY" \
      -d '{"projectId": "proj_123"}'
    ```
  </Tab>
  <Tab title="HTML" icon="code">
    Get a complete HTML bundle with styles and assets.
    
    ```
    curl -X POST https://api.example.com/v1/export/html \
      -H "Authorization: Bearer YOUR_API_KEY" \
      -d '{"projectId": "proj_123", "theme": "dark"}'
    ```
  </Tab>
</Tabs>

## Embedding External Content

Embed content from external sources like Figma prototypes or YouTube videos directly into U17g pages.

<Expandable title="Advanced Embedding Options" default-open="false">
  Use iframe embeds for interactive content. Ensure the source allows embedding via CORS.
  
  ```
  <iframe
    src="https://www.youtube.com/embed/VIDEO_ID"
    width="100%"
    height="400"
    frameborder="0"
    allowfullscreen
  ></iframe>
  ```
</Expandable>

<Callout kind="info">
  Test embeds in preview mode to confirm they render correctly across devices.
</Callout>
