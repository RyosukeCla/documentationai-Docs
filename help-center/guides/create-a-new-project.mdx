---
title: Creating a New Project
description: How to make a new project in U17g.
---

## Steps

<Steps>
  <Step title="Navigate to Projects">
    ````
    Go to the U17g dashboard and select "My Projects" from the sidebar.
    ````
  </Step>
  <Step title="Start a new project">
    ````
    Click "Create Project".
    Enter a name and (optionally) a description.
    Save to finish.
    ````
  </Step>
  <Step title="Add documentation pages">
    ````
    Within your new project, click "+ New Page" to add docs.
    Start writing!
    ````
  </Step>
</Steps>
