title: Cannot Access Documentation
description: Fix problems opening your U17g documentation.
---

## Issue

- Docs page won't load
- Get a 404 or access denied

## Quick Fix

<Steps>
  <Step title="Check login status">
    ````
    Make sure you are logged into U17g.
    Logout and log in again if needed.
    ````
  </Step>
  <Step title="Verify permissions">
    ````
    Ensure your account has access to the project docs.
    Ask the project admin to add you if not.
    ````
  </Step>
</Steps>
