## Introduction

To start using U17g, create your first project and explore the workspace.

## Steps

1. Visit `https://app.u17g.com`
2. Sign up with your email
3. Click "Create Project"
4. Enter a name and description
5. Start adding documentation pages

U17g lets you manage and organize all your docs in one place.
