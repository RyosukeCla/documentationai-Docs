## Prerequisites

<Callout kind="info" title="Before You Begin">
You need a modern web browser like Chrome, Firefox, or Safari. No software installation required. An email address for account verification is essential.
</Callout>

Ensure you have access to `https://app.u17g.com` to sign up.

## Sign Up and Log In

Follow these steps to create your U17g account and access the dashboard.

<Steps>
  <Step title="Visit U17g" icon="globe">
    Open your browser and navigate to `https://app.u17g.com`.
  </Step>
  <Step title="Create Account" icon="user-plus">
    Click **Sign Up** and enter your email, password, and workspace name. Verify your email via the confirmation link sent to your inbox.
  </Step>
  <Step title="Log In" icon="log-in">
    Return to the login page, enter your credentials, and select your workspace. You land on the main dashboard.
  </Step>
</Steps>

## Create Your First Page

Start building your docs with these methods.

<Tabs>
  <Tab title="Blank Page" icon="plus">
    <Steps>
      <Step title="New Page" icon="file-plus">
        Click **New Page** in the sidebar. Choose a title like "My First Doc".
      </Step>
      <Step title="Edit Content" icon="edit-3">
        Use the visual editor to add headings, lists, and text. Preview changes live.
      </Step>
      <Step title="Publish" icon="upload">
        Hit **Publish** to make it live. Share the URL with your team.
      </Step>
    </Steps>
  </Tab>
  <Tab title="Markdown Import" icon="code">
    Import existing Markdown files seamlessly.

    <CodeGroup tabs="CLI,Direct Upload">
      ```bash
      # Use our CLI to push Markdown
      npm install -g @u17g/cli
      u17g push ./docs/my-page.md --workspace your-workspace
      ```
      ```javascript
      // Or via API
      const response = await fetch('https://api.u17g.com/v1/pages', {
        method: 'POST',
        headers: {
          'Authorization': `Bearer ${YOUR_API_KEY}`,
          'Content-Type': 'application/json'
        },
        body: JSON.stringify({
          title: 'Imported Page',
          content: '# Hello World\nThis is markdown content.'
        })
      });
      ```
    </CodeGroup>
  </Tab>
</Tabs>

## Basic Workspace Setup

Customize your space for better organization.

<Steps>
  <Step title="Add Members" icon="users">
    Go to **Settings > Members** and invite collaborators by email.
  </Step>
  <Step title="Set Permissions" icon="shield">
    Assign roles like Editor or Viewer to control access.
  </Step>
  <Step title="Integrate Tools" icon="plug">
    Connect GitHub or Slack via **Settings > Integrations**.
  </Step>
</Steps>

<Expandable title="Advanced: Custom Domain" default-open="false">
Configure a custom domain in **Settings > Domains**. Add a CNAME record pointing to `pages.u17g.com`.
</Expandable>
